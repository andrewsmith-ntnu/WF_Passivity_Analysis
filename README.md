Wind farm impedance aggregation and analysis

This repository contains MATLAB scripts and functions for modeling, aggregating and
analyzing wind turbine (WT) impedances and stability using different generator
and converter models.

Contents
- MATLAB functions and scripts (.m) for steady-state, matrices, and impedance computation
- Simulink models (.slx) for onshore/offshore setups
- Blackboxed implementations in `Blackboxed/`

Quick Git setup
1) Set your Git user name and email globally:
   git config --global user.name "Your Name"
   git config --global user.email "you@example.com"

2) To set them only for this repository (overrides global):
   git config user.name "Your Name"
   git config user.email "you@example.com"

3) Verify your configuration:
   git config --global --list
   git config user.email
